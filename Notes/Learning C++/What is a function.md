# What is a function?

- Function is just a black box which computes the output from any given input
- We can create our own functions to do specific tasks
- Some are however provided to use by the C++ language
- Examples:
	- `string` library:
		- `string.length()`
	- `iomanip` library:
		- `Setw()`
		-  `Setfill()`
		-  `Setprecision()`
	-  `cctype`
		-  `isdigit`
		-  `isalpha`
		-  `isupper`
		-  `islower`
	-  `cstdlib`
		-  `rand`
		-  `srand`
