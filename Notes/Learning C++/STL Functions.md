- Functor
	- is nothing but a class with a operator overload with a function called operator

- arithmetic functors
	- defined in `functional` header
	- `plus`
	- `minus`
	- `multiplies`
	- `divides`
	- `modulus`
	- `negate`

- relational functors
	- defined in `functional` header
	- `greater`
	- `less`
	- `greater_equal`
	- `less_equal`
	- `equal_to`
	- `not_equal`

- logical
	- `logical_and`
	-  `logical_or`
	-  `logical_not`
